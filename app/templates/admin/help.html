MiniStatus

MiniStatus is the read-only operator console for the polymarket-alerts engine. It aggregates alerts_log.csv and related artifacts from your configured POLYMARKET_DATA_PATH.

Where to start

  • Live — portfolio P/L, resolved list, export.
  • Scorecard — per-strategy resolved performance.
  • AI Simulation — AI gate stats (safe scope toggle).
  • Tools — Open Positions, Analytics, mirrors, dev tools (URL /polymarket/ops).

Settings

Use the sidebar for Change password. Default login is set in your .env file.
