What does this tab do?

The Lifecycle tab evaluates shadow and active strategies using resolved results. It decides whether strategies are building enough evidence to trust (promote) or should be retired (kill).

If you don’t see data yet, generate it from the polymarket-alerts repo: run the lifecycle export and refresh the page.
