Quick Start

Get MiniStatus running in minutes with our simple installation script.

# Clone the repository
git clone https://github.com/LieAndSmile/MiniStatus-MVP.git
cd MiniStatus-MVP

# Run the installer
sudo bash scripts/install.sh

The installer will set up everything automatically. Access the dashboard at http://YOUR_SERVER_IP:5000
